# ESS Website

A modern HTML website built with Tailwind CSS and Vite.

## Getting Started on Mac

### Prerequisites

Install Node.js if you don't have it:
```bash
brew install node
```

### Run the Project

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The site will open in your browser at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

Built files will be in the `dist/` folder.

## Technologies

- HTML5
- Tailwind CSS 3
- Vite
- Alpine.js
